# Calculator

## Anforderungen

    -   Grundrechenarten - Opperationen

    -   Clear : ganze - Eingabe löschen

    -   *    : Opperator - Multiplikation
    -   +    : Opperator - Addition
    -   -    : Opperator - Subtraktion
    -   -    : Opperator - Ergebnis (result)
    -   -    : Opperator - Ergebnis (result)
